# The hosted chat page.

If you don't want a floating bubble, you can use the **hosted chat page** instead — a full-page chat surface Tenlo hosts for you.

## URL format

```
https://app.tenloai.com/chat/YOUR-CHATBOT-ID
```

## Use cases

- Link to it from a support email — *"Click here to chat with our AI assistant"*
- Embed it as an iframe on a custom support page
- Redirect to it from a "Contact us" link
- Share the URL on social or in a help-centre footer

## Behaviour

The hosted page uses the same KB, the same handoff rules, and the same brand voice as the embedded widget. Just a different surface.

Unlike the [Sales Assistant (P05)](/sales-assistant), which has no hosted page equivalent, the Support Chatbot gives you both options.
